# Tucker Family Site Deployment

Static family-history site with a simple client-side family password screen.

- Password: `9898`
- Main page: `index.html`

This is intentionally simple family-level privacy, not high-security.

## Deploy to Cloudflare Pages manually

1. Go to Cloudflare Dashboard → Workers & Pages → Pages → Create application → Upload assets.
2. Upload either this folder or `tucker-family-site.zip`.
3. Share the generated `*.pages.dev` link.
